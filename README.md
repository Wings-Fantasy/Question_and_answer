# 软件功能
在软件内添加关键词与对应的答案后即可使用，主页即问题查找页，输入问题后点击提问即可。<br>

# 注意事项
软件内提问时，问题中必须包含一个关键词才能找到答案<br>

# 已知BUG
1.问题中包含多个关键词时，不能准确的找到答案<br>
